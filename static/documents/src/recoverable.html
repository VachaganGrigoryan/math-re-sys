ՎԵՐԱԿԱՆԳՆԵԼԻ ՀԱՄԱԿԱՐԳԵՐԻ ՀՈՒՍԱԼԻՈՒԹՅԱՆ ՀԱՇՎԱՐԿԸ

Վերականգնվող տարրերի հուսալիության ցուցանիշները բնութագրում են օբյեկտի հուսալիությունը ոչ միայն մինչև նրա առաջին խափանումը, այլև խափանումների միջև ընկած ժամանակահատվածում: Դիտարկենք ըստ հուսալիության հաջորդաբար միացված վերականգնվող համակարգ։ Ընդհանուր դեպքում, եթե ունենք n ենթահամակարգ և n բրիգադ, ապա համակարգի աշխատունակ վիճակում գտնվելու ստացիոնար հավանականությունը հավասարվում է պատրաստականության գործակցին, որը որոշվում է հետևյալ բանաձևով.

{%22 :; P_1=\frac{\mu^n}{(\lambda+\mu)}^n=K_p^n %}


Պատրաստականության ֆունկցիան`{%16 :; K_p(t) %}-ն այն բանի հավանականությունն է, որ համակարգը պատրաստ է աշխատել կամայական t ժամանակ, որը որոշվում է հետևյալ բանաձևով.

{%22 :; K_p(t)=\frac{\mu}{\lambda+\mu}+\frac{\mu}{\lambda+\mu}e^{-(\lambda+\mu)}t %}


Քննարկենք ընդհանուր մշտական պահուստավորված համակարգի հուսալիության հաշվարկը: Երբ ունենք {%16 :; n=m+1 %} համակարգ (1 հիմնական ու m պահուստային) և 1 վերանորոգող բրիգադ, {%16 :; K_p(t) %} որոշվում է հետևյալ բանաձևով՝

{%22 :; K_p=\frac{\sum_{j=0}^{m} \rho^j}{\sum_{j=0}^{m+1} \rho^j} %}, որտեղ՝ {%22:; \rho=\frac{\lambda}{\mu} %}


Ընդհանուր դեպքում, երբ ունենք {%16 :; n=m+1 %} համակարգերի և n վերանորոգող բրիգադներ, {%16 :; K_p(t) %} որոշվում է հետևյալ բանաձևով՝

{%22 :; \frac{\sum_{j=0}^{m} \rho^j/j!}{\sum_{j=0}^{m+1} \rho^j/j!} %} կամ {%22 :; K_p=1-\frac{\rho^{m+1}}{(1+\rho)^{m+1}} %}


Այժմ քննարկենք վերականգնվող համակարգերի հուսալիության հաշվարկը ընդհանուր փոխարինումով պահուստավորման դեպքում։ Ընդհանուր դեպքում, երբ ունենք {%16 :; (m+1) %} համակարգ և 1 բրիգադ, պահուստավորված միացության պատրաստականության գործակիցը որոշվում է հետևյալ բանաձևով՝

{%22 :; K_p=\frac{\sum_{i=0}^{m} x^i}{\sum_{i=0}^{m+1} x^i} %}, որտեղ՝ {%22:; x=\frac{\mu}{\lambda} %}


{%16 :; (m+1) %} համակարգի և նույնքան բրիգադների դեպքում, պահուստավորված միացության պատրաստականության գործակիցը կորոշվի հետևյալ բանաձևով՝

{%22 :; K_p=\frac{\sum_{i=0}^{m} x^i/i!}{\sum_{i=0}^{m+1} x^i/i!} %}
